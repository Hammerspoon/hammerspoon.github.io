hs.pasteboard

Inspect/manipulate the system clipboard

This module is based partially on code from the previous incarnation of Mjolnir by Steven Degutis.

API Overview

  • Functions - API calls offered directly by the extension
    • changeCount
    • getContents
    • setContents

API Documentation

Functions

changeCount

Signature hs.pasteboard.changeCount() -> number
Type Function
Description

Gets the number of times the pasteboard owner has changed

Parameters:

  • None

Returns:

  • A number containing a count of the times the pasteboard owner has changed

Notes:

  • This is useful for seeing if the pasteboard has been updated by another process

getContents

Signature hs.pasteboard.getContents() -> string or nil
Type Function
Description

Gets the contents of the pasteboard

Parameters:

  • None

Returns:

  • A string containing the contents of the pasteboard, or nil if an error occurred

setContents

Signature hs.pasteboard.setContents(contents) -> boolean
Type Function
Description

Sets the contents of the pasteboard

Parameters:

  • contents - A string to be placed in the pasteboard

Returns:

  • True if the operation succeeded, otherwise false
