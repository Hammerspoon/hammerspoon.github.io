hs.battery.watcher

Watch for battery/power state changes

This module is based primarily on code from the previous incarnation of Mjolnir by Steven Degutis.

API Overview

  • Functions - API calls offered directly by the extension
    • stop
  • Constructors - API calls which return an object, typically one that offers API methods
    • new
  • Methods - API calls which can only be made on an object returned by a constructor
    • start

API Documentation

Functions

stop

Signature hs.battery.watcher:stop() -> self
Type Function
Description

Stops the battery watcher's fn from getting called until started again.

Constructors

new

Signature hs.battery.watcher.new(fn) -> watcher
Type Constructor
Description

Creates a battery watcher that can be started. When started, fn will be called each time a battery attribute changes.

Methods

start

Signature hs.battery.watcher:start() -> self
Type Method
Description

Starts the battery watcher, making it so fn is called each time a battery attribute changes.
