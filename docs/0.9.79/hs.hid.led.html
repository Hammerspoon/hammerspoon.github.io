set
Signature hs.hid.led.set(name, state) -> bool
Type Function
Description

Assigns HID LED to the desired state Note that this function controls the LED state only, to modify capslock state, use hs.hid.capslock.set

Parameters:

  • name - LED name: "caps", "scroll" or "num"
  • state - A boolean indicating desired state

Returns:

  • true if success, false if error
