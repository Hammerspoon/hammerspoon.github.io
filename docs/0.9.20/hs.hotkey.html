hs.hotkey

Create and manage global keyboard shortcuts

API Overview

  • Constructors - API calls which return an object, typically one that offers API methods
    • bind
    • new
  • Methods - API calls which can only be made on an object returned by a constructor
    • disable
    • enable

API Documentation

Constructors

bind

Signature hs.hotkey.bind(mods, key, pressedfn, releasedfn) -> hotkey
Type Constructor
Description

Shortcut for: return hs.hotkey.new(mods, key, pressedfn, releasedfn):enable()

new

Signature hs.hotkey.new(mods, key, pressedfn, releasedfn = nil) -> hotkey
Type Constructor
Description

Creates a new hotkey that can be enabled.

The mods parameter is case-insensitive and may contain any of the following strings: "cmd", "ctrl", "alt", or "shift".

The key parameter is case-insensitive and may be any string value found in hs.keycodes.map

The pressedfn parameter is the function that will be called when this hotkey is pressed.

The releasedfn parameter is the function that will be called when this hotkey is released; this field is optional (i.e. may be nil or omitted).

Methods

disable

Signature hs.hotkey:disable() -> self
Type Method
Description

Disables the given hotkey; does not remove it from hs.hotkey.keys.

enable

Signature hs.hotkey:enable() -> self
Type Method
Description

Registers the hotkey's fn as the callback when the user presses key while holding mods.
