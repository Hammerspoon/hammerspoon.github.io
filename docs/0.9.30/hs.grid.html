hs.grid

Move/resize windows within a grid

Usage: local grid = require "hs.grid"

The grid is an partition of your screen; by default it is 3x3, i.e. 3 cells wide by 3 cells tall.

Grid cells are just a table with keys: x, y, w, h

For a grid of 2x2:

  • a cell {x=0, y=0, w=1, h=1} will be in the upper-left corner
  • a cell {x=1, y=0, w=1, h=1} will be in the upper-right corner
  • and so on...

API Documentation

Variables

GRIDHEIGHT

Signature hs.grid.GRIDHEIGHT = 3
Type Variable
Description

The number of cells high the grid is.

GRIDWIDTH

Signature hs.grid.GRIDWIDTH = 3
Type Variable
Description

The number of cells wide the grid is.

MARGINX

Signature hs.grid.MARGINX = 5
Type Variable
Description

The margin between each window horizontally.

MARGINY

Signature hs.grid.MARGINY = 5
Type Variable
Description

The margin between each window vertically.

Functions

adjustFocusedWindow

Signature hs.grid.adjustFocusedWindow(fn)
Type Function
Description

Passes the focused window's cell to fn and uses the result as its new cell.

adjustHeight

Signature hs.grid.adjustHeight(by)
Type Function
Description

Increases the grid by the given number of cells; may be negative

adjustWidth

Signature hs.grid.adjustWidth(by)
Type Function
Description

Widens the grid by the given number of cells; may be negative

get

Signature hs.grid.get(win)
Type Function
Description

Gets the cell this window is on

maximizeWindow

Signature hs.grid.maximizeWindow()
Type Function
Description

Maximizes the focused window along the given cell.

pushWindowDown

Signature hs.grid.pushWindowDown()
Type Function
Description

Moves the focused window to the bottom half of the screen.

pushWindowLeft

Signature hs.grid.pushWindowLeft()
Type Function
Description

Moves the focused window one cell to the left.

pushWindowNextScreen

Signature hs.grid.pushWindowNextScreen()
Type Function
Description

Moves the focused window to the next screen, using its current cell on that screen.

pushWindowPrevScreen

Signature hs.grid.pushWindowPrevScreen()
Type Function
Description

Moves the focused window to the previous screen, using its current cell on that screen.

pushWindowRight

Signature hs.grid.pushWindowRight()
Type Function
Description

Moves the focused window one cell to the right.

pushWindowUp

Signature hs.grid.pushWindowUp()
Type Function
Description

Moves the focused window to the top half of the screen.

resizeWindowShorter

Signature hs.grid.resizeWindowShorter()
Type Function
Description

Resizes the focused window so its height is 1 grid count less.

resizeWindowTaller

Signature hs.grid.resizeWindowTaller()
Type Function
Description

Resizes the focused window so its height is 1 grid count higher.

resizeWindowThinner

Signature hs.grid.resizeWindowThinner()
Type Function
Description

Resizes the focused window's right side to be one cell thinner.

resizeWindowWider

Signature hs.grid.resizeWindowWider()
Type Function
Description

Resizes the focused window's right side to be one cell wider.

set

Signature hs.grid.set(win, grid, screen)
Type Function
Description

Sets the cell this window should be on

snap

Signature hs.grid.snap(win)
Type Function
Description

Snaps the window into a cell
