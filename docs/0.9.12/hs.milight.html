hs.milight

Simple controls for the MiLight LED WiFi bridge (also known as LimitlessLED and EasyBulb)

maxBrightness

hs.milight.maxBrightness

Specifies the maximum brightness value that can be used

minBrightness

hs.milight.minBrightness

Specifies the minimum brightness value that can be used

new

hs.milight.new(ip[, port]) -> bridge

Creates a new bridge object, which will be connected to the supplied IP address and port

Parameters:

  • ip - IP address of the MiLight WiFi bridge device. For convenience this can be the broadcast address of your network (e.g. 192.168.0.255)
  • port - UDP port to talk to the bridge on. Defaults to 8899

Returns:

  • bridge object to use with other API methods

zoneOff

hs.milight.zoneOff(zone) -> bool

Turns off the specified zone

Parameters:

  • zone - 0 for all zones, 1-4 for zones one through four

Returns:

  • True if the command was sent correctly, false if not

send

hs.milight:send(cmd, value) -> bool

Sends a command to the bridge.
Note: This is a low level command, you typically should use a specific method for the operation you want to perform

Parameters:

  • cmd - A command from the hs.milight.cmd table
  • value - An optional value, if appropriate for the command (defaults to 0x00)

Returns:

  • True if the command was sent, false if an error occurred

zoneBrightness

hs.milight:zoneBrightness(zone, value) -> bool

Sets brightness for the specified zone

Parameters:

  • zone - 0 for all zones, 1-4 for zones one through four
  • value - Brightness level to set, between 0 and 25

Returns:

  • The value that was sent to the WiFi bridge, or -1 if an error occurred
