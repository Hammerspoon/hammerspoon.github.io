hs.mouse

Inspect/manipulate the position of the mouse pointer.

This module is based primarily on code from the previous incarnation of Mjolnir by Steven Degutis.

API Documentation

Functions

get

Signature hs.mouse.get() -> point
Type Function
Description

Returns the current location of the mouse on the current screen as a point.

set

Signature hs.mouse.set(point)
Type Function
Description

Moves the mouse to the given location on the current screen.
